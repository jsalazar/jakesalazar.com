Liferay Portal Development

EduTEKS is a Liferay service provider and specializing in Liferay development. Our Certified Liferay experts have many years of involvement with developing and delivering fully featured enterprise portal solutions using the Liferay Portal Platform. As a Liferay offshore service provider, we facilitate a plethora of services ranging from consulting and development to support, using Liferay dxp, Liferay 6.x, and Liferay 7.x.


Liferay Development / Customization
  • Design architecture
  • Implement out-of-the-box portlets, modules, features
  • Module/Portlet development using JSR 168 and 286 standards
  • Liferay theme customization
  • Liferay template customization
  • Design and development of custom workflows
  • Develop and design of custom security providers and authentication models
  • Configure Liferay for a clustered environment
  • Implement SOAP and REST web services using Liferay Portal
  • Customization using Liferay events

Liferay Consulting
  • Recommendation of hardware and software environments
  • Liferay Portal Implementation
  • Authentication, Single sign-on with LDAP/AD, etc
  • Liferay Customization & Deployment
  • Liferay Portal Integration with custom application
  • Liferay Module/Portlet development & support services
  • Assistance with migration from any portal to Liferay portal
  • Analysis of client’s requirements and scope of work
  • Create Proof-of-Concepts (POC) for Liferay Portal

Liferay Support & Maintenance
  • Monitor your Liferay portal constantly which eliminates downtime of your business
  • Provide bug fixes and feature enhancements for custom developed modules as well as OOTB Liferay modules
  • Quick and easy solutions for critical/complex Liferay portal problems
  • Assurance for smooth and efficient functioning of your production Liferay portal
  • Professional expertise in releasing bug fixes and feature enhancements by following standard deployment and verification processes
  • Monitor product usages by various tools
  • Liferay upgrades

Liferay Migration
  • Migration from Microsoft SharePoint, IBM WebSphere, and other portals to Liferay Portal
  • PHP Website website ( Drupal, WordPress, Joomla) to Liferay
  • Static website to Liferay Portal
  • PSD to Liferay Portal
  • JSP, Spring-based portal to Liferay Portal

Liferay Integration
  • 3rd-party integration (authentication, SSO, external apps and services)
  • Payment Gateway Integration using SDK/Rest API
  • Custom Java-based software integration
  • Integration with Alfresco
  • Enterprise Software Integration
  • Social Media Integration with Liferay

Liferay Performance Tuning
  • Portal properties configuration
  • JVM Memory tuning
  • Optimize server thread Pool and JDBC connection pool
  • Disable unnecessary filters
